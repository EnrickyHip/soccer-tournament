# Shuffle Array
Simplest shuffle array algorithm.

## Installation

```shell
$ npm install easy-shuffle
```

## Usage

```js
const shuffle = require('easy-shuffle');

const array = [1, 2, 3, 4, 5];

const shuffledArray = shuffle(array);

console.log(shuffledArray) // [ 4, 1, 2, 5, 3 ];
```
